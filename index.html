Javascript এর কাজ

Javascript অন্যান্য programing language যেমন C++ বা Java language এর মতো শুরু করার আগে Compiled করা লাগে না। কারন এই language টি একটি interpreted ল্যাঙ্গুয়েজ। Javascript single threaded and single concurrent programing language অর্থাৎ এটি এক সময় শুধুমাত্র একটি code এর কাজ করতে পারে। Javascript তৈরি করা হয়েছে মুলত web developer দের সুবিধার জন্যে, যেন তারা কোডকে খুব সহজেই interactive ভাবে web page এ চালু করতে পারে। Javascript সব ধরনের web browser এবং সব ধরনের device এ চলতে পারে।
